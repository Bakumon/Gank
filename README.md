## 介绍

「有干货」是一款注重体验的 [Gank.io](http://gank.io) 第三方安卓客户端，首页即可随机查看妹子，还能浏览各种类型（App、Android、iOS、前端、瞎推荐、拓展资源等）的干货数据，搜索界面提供妹子福利查看大图功能。App 整体秉承简洁风格。
